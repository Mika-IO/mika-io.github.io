即时求任意数字的平方根

平方根是最常用的数学运算之一。从勾股定理到标准差，从二次方程到金融公式，平方根无处不在。输入任意非负数，其平方根会立即显示，同时显示平方值（n²）以供参考。

什么是平方根？

一个数 n 的平方根是这样一个值：它与自身相乘等于 n。写作 √n 或 n^(1/2)。例如：

  • √9 = 3，因为 3 × 3 = 9
  • √25 = 5，因为 5 × 5 = 25
  • √2 ≈ 1.4142135...（一个无理数）
  • √100 = 10

每个正数都有两个平方根：一个正数和一个负数。正数根称为主平方根。零的平方根是零。

完全平方数

完全平方数是指有整数平方根的数：

  • 1, 4, 9, 16, 25, 36, 49, 64, 81, 100, 121, 144, 169, 196, 225...

在算术、几何和代数中识别完全平方数很有用。

无理数平方根

大多数数没有精确的整数平方根。√2、√3、√5、√6、√7 以及许多其他的数都是无理数——它们的小数展开是无穷且不循环的。这些值在几何学中很重要：一个单位正方形的对角线长度为 √2 ≈ 1.41421356...

应用领域

勾股定理：直角三角形的斜边 c 满足 c = √(a² + b²)。计算时需要取平方根。

标准差：一个数据集的标准差是其方差的平方根。取平方根使度量回到数据的原始单位。

求根公式：方程 ax² + bx + c = 0 的解为 x = (-b ± √(b² - 4ac)) / 2a。平方根下的判别式 b² - 4ac 决定了解是否为实数。

距离公式：两点 (x₁, y₁) 和 (x₂, y₂) 之间的距离为 √((x₂-x₁)² + (y₂-y₁)²)。

金融模型：收益率的标准差衡量投资风险。投资组合数学大量使用平方根。

物理学：波动方程、能量计算以及许多物理关系都涉及平方根。

负数与虚数

负数的平方根不是实数。在复数系统中，√(-1) 被定义为虚数单位 i。复数的形式为 a + bi，广泛应用于电气工程、量子力学和许多其他领域。

用手估算平方根

在拿起计算器之前，了解如何粗略估算平方根很有帮助。找到离你数字最近的两个完全平方数：对于 √50，最近的完全平方数是 49（√49 = 7）和 64（√64 = 8），所以答案一定在 7 和 8 之间，而且由于 50 更接近 49，答案应该接近 7 但稍高一些——真值 7.07 证实了这个直觉。这种夹逼技巧是一种有用的验证手段，当你想确认计算器的结果大致正确时很方便，也正是心算和初等代数课程旨在培养的那种估算能力。

牛顿法——幕后的算法

现代计算机不是通过查表来求平方根的；它们使用快速迭代方法来计算，其中最著名的是牛顿法（也称巴比伦法，因为古代巴比伦数学家就已经知道了它的一个版本）。从一个粗略的猜测开始，每一步都使用公式来精炼估计：下一个猜测 = (猜测 + 数值 / 猜测) / 2。用于从初始猜测 3 开始求 √10：(3 + 10/3)/2 ≈ 3.1667，然后 (3.1667 + 10/3.1667)/2 ≈ 3.1623，这已经非常接近真值 3.16228。每次迭代大约将正确位数翻倍，这就是为什么现代处理器只需几步就能以满精度计算平方根。

平方与平方根作为互逆运算

平方和取平方根互为逆运算，这就是为什么本计算器同时显示两者：n² 的平方根返回 n（对于非负 n），√n 的平方返回 n。这种互逆关系在整个代数中都是基础性的——这正是涉及平方项的方程（如二次公式或勾股定理）通过在适当时机两边取平方根来求解的原理。

Square root FAQ

What is a square root?
The square root of a number n is a value that, when multiplied by itself, gives n. For example, √9 = 3 because 3 × 3 = 9. Every positive number has two square roots: a positive and a negative one.
What about negative numbers?
The square root of a negative number is not a real number — it is an imaginary number. For example, √(-1) = i in the complex number system.
What are perfect squares?
A perfect square is a number whose square root is a whole integer: 1, 4, 9, 16, 25, 36, 49, 64, 81, 100... are perfect squares.
