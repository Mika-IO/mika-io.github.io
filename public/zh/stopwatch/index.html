    一款精确的秒表，就在你的浏览器中

    准确计时曾经意味着需要随身带一个专用秒表。现在，任何带有浏览器的设备都可以胜任这个工作。这款在线秒表完全在你的浏览器中使用高精度计时器运行，支持记录分段成绩，并以分钟、秒和毫秒显示经过的时间。无需下载或安装任何东西。

    如何使用

    按下开始按钮开始计时。显示画面会实时展示分钟、秒和千分之一秒。在秒表运行时，按分段按钮可以记录当前经过的时间。分段记录列表会累积显示在画面下方，编号排列以便比较。按停止按钮暂停秒表，不会丢失已记录的时间或分段记录。再按开始按钮可以从暂停处继续。按重置按钮清除所有数据并归零。

    精度与浏览器计时器

    现代浏览器通过 Performance API 提供了高精度计时器，可以以毫秒的分数来测量时间。这比旧版 Date.now() 方法的 1 毫秒分辨率精确得多。实际上，在大多数设备上，计时器的精度远优于 1 毫秒，对于运动计时、烹饪、演讲或任何日常使用来说绰绰有余。如需在竞赛中使用法律认证的计时，请始终使用批准的专用计时设备。

    一个值得了解的局限性是：如果你切换到其他浏览器标签页，浏览器可能会在后台限制 JavaScript 的运行以节省资源。这可能导致秒表略微落后于实际经过的时间。保持秒表标签页处于活动状态以获得最佳精度。

    常见用途

    运动员使用秒表来为跑步、游泳圈数、自行车间歇或任何需要比较分段成绩的重复性运动计时。分段记录帮助你查看自己是在保持配速、进步还是在掉速。教练可以为多名运动员依次计时并比较结果。

    在厨房里，当你同时在处理多道需要不同时间的菜品时，秒表比倒计时器更灵活。你在每道菜入锅时开始计时，记下分段时间来了解每道菜已经烹饪了多久。

    演讲和演示也会受益于一个运行中的计时器。准确地知道你已进行了四分三十秒，可以帮助你判断是扩展还是压缩某个部分以达到目标时长。

    科学家和学生在实验中利用分段功能记录数据点，而不必为下一次试验重新启动计时器。

    触屏与键盘友好

    开始、分段和重置按钮支持鼠标点击和手指触控。布局在手机和平板电脑上也能清晰展示。

    为什么浏览器秒表真的够精确

    值得理解为什么一个运行在网页中的免费工具可以被信任用于不仅仅是随意的计时。现代浏览器通过 Performance API 提供了高精度计时器，区别于老旧、粗糙的 Date.now() 方法，能够以毫秒的分数而非整毫秒来测量时间。在实际使用中，这意味着计时器在一次普通计时过程中的漂移远小于 1 毫秒，足以满足运动、烹饪、演讲和实验室工作——唯一需要注意的是，浏览器会故意减慢后台标签页中的 JavaScript 以节省电池和 CPU，所以保持秒表标签页处于活动状态是保持其精度的唯一习惯。

    分段成绩与圈速的区别

    计时爱好者有时区分"圈速"（仅该段的独立耗时）和"分段成绩"（从开始到该检查点的累计经过时间）。本工具记录的是分段成绩——每条记录的时间是你按下分段按钮时的累计总时间——这在休闲计时中是更常见的惯例，也符合大多数人在看一眼运行中的秒表时期望看到的内容。如果你需要单圈耗时而非累计分段成绩，将每条记录的时间减去前一条即可。

    秒表与手机内置计时器

    每部现代智能手机都自带秒表应用，所以有人会问为什么还要使用基于浏览器的秒表。坦诚的答案是便利性而非更高的精度：在共享电脑上、在视频通话中、或者在一个已经运行在浏览器标签页的工作流中，打开一个网页比解锁手机并找到正确的应用更快，而且来自 Performance API 的底层计时精度与原生应用在非竞赛的日常使用中达到的水平相当。

    同时为多项活动计时

    某些情况需要同时跟踪多件事——多道以不同开始时间烹饪的菜品，或者一场接力赛中的多个赛段。由于本秒表维护一个运行总计加一个分段列表，同时处理多项活动的实用方法是在不同的浏览器标签页中分别打开工具，每个对应一项活动，各自在对应活动开始时独立启动。

    中途重置

    如果你在保存所需的分段成绩之前不小心按了重置，没有撤销功能——计数会立即归零。对于分段记录确实很重要的任何场景，值得在产生的同时记下关键的分段成绩，而不是完全依赖屏幕上的列表保留到最后。

    隐私与随时可用

    任何数据都不会被发送到任何地方，秒表完全在你的浏览器中运行，页面加载完成后即可离线工作，无需账户——随时重新加载页面即可将一切重置为零。

    Stopwatch FAQ

    How accurate is the browser stopwatch?
    The stopwatch uses the browser's high-resolution performance timer, which is accurate to well under a millisecond in normal conditions. For official timing where exact precision is required, use a certified timing device.
    Will it stop if I switch tabs?
    Browsers may throttle JavaScript in background tabs, which could affect accuracy if you switch away. Keep the stopwatch tab active for best results.
    Can I use this on a mobile phone?
    Yes, the stopwatch is fully touch-compatible and works on smartphones and tablets.
