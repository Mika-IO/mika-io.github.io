颜色对比检查器的用途是什么

选择前景色和背景色以检查对比度和 WCAG 2.1 辅助功能合规性。

这是简短的版本，对于许多访问者来说，这是他们唯一需要的部分 - 该工具位于此页面的顶部，它可以立即运行，并且不需要阅读下面的任何内容。本页的其余部分适用于仅靠答案还不够的时候：当您想知道生成器实际上是如何执行的、它在哪里不再可靠以及获得结果后如何处理时。

如何使用

控制措施故意很少。您将看到 2 个控件：

  • 文字颜色
  • 背景颜色

填写适用于您的内容，结果就会更新。就在任何地方提交而言，没有任何内容是强制性的 - 如果您改变主意，请编辑值并阅读新答案。没有保存步骤，没有确认对话框，也没有办法永久出错，因此尝试几种变体来查看结果如何移动不仅是允许的，而且通常是使用此类工具的最有用的方法。

一切都在您自己的浏览器中运行。您输入的任何内容都不会被上传，访问之间不会存储任何内容，无需创建帐户，也无需拒绝跟踪。此页面加载后就不再需要网络，因此即使您的连接断开，它也能继续工作。

为什么存在这样的页面

设计工作从快速反馈循环中获得的好处远远超过第一次尝试时的精确度。能够推动一个值并立即看到结果，可以将某种事情不对劲的模糊感觉转变为你可以捍卫的具体决定。

网络上有很多工具可以通过首先要求提供电子邮件地址、cookie 决策和四秒钟的广告来解决三十秒的问题。这笔交易对某些人来说是有意义的，但对只想要数字的人来说却没有意义。此页面采取相反的立场：没有注册，没有插页式广告，没有追加销售，一旦你得到了你想要的东西就没有理由离开。

结果告诉你什么，没有告诉你什么

输出是标准 CSS，您可以直接粘贴到样式表中 — 没有框架、没有专有格式、没有导出步骤。您看到的渲染内容是由您复制的相同声明生成的，因此预览和代码库之间不需要转换任何内容。

明确界限是值得的，因为工具通常会受到超出其应得信任的信任。这个输入是诚实的：给它准确的值，它就会给你一个准确的生成器。它无法告诉您所提供的值是否正确，您是否提出了正确的问题，或者您应用答案的上下文是否具有输入从未捕获的重要内容。这种判断由你来决定，这才是真正重要的部分。

它适合真实工作流程的地方

像这样的工具很少能完成工作。更常见的是，颜色对比度检查器是更大的事情中间的一个步骤：您需要这个数字，以便您可以继续您实际正在做的事情。这决定了页面的构建方式。它打开速度很快，不会打断，不会要求你做出任何承诺，当你得到你想要的东西时，它就会消失。

这也意味着您应该随意重复使用它，而不是小心翼翼地使用它。第二次尝试没有任何成本，第一次猜测错误也不会受到惩罚，因此获得好答案的最快途径通常是尝试近似答案，查看结果并进行调整。人们倾向于认为工具的每次使用都是昂贵的，这是从使其昂贵的软件继承下来的习惯。这个没有。

值得避免的常见错误

第一个是信任您未检查其输入的结果。工具无法知道您打算输入一个数字并输入另一个数字，并且它会以与正确答案完全相同的置信度呈现错误答案。在依赖输出之前读回您的输入 - 只需两秒钟即可发现大部分问题。

第二种是将一个数字视为对从未被问过的问题的答案。此页面上的每个生成器都回答了一些狭窄且具体的问题。将其应用于附近但不同的问题是人们最终被一直运行良好的工具误导的最常见方式。如果您不确定您遇到的问题是否与工具回答的问题相符，那么这种不确定性就是有用的信号，而不是障碍。

在任何设备上离线工作

该页面是一个独立的文档。没有可供下载的框架，没有从第三方获取的字体，也没有在后台触发的分析信标，这就是为什么它在连接不良时加载速度很快，以及为什么在您偶尔返回的选项卡中保持打开状态不需要花费任何费用。它在手机、平板电脑和台式机上的工作方式相同；布局会适应，但行为不会。

因为实际工作中没有涉及服务器，所以也没有要停机的服务，没有要达到的速率限制，也没有从其他地方推出的行为的安静变化。你加载的就是运行的。如果您想确定这一点，请打开页面源代码 - 整个实现都以可读的形式存在，这是很少有此类工具可以做到的声明。

关于准确性的说明

结果使用标准双精度算术进行计算，并显示到合理数量的位置，而不是硬件可以生成的每个数字。显示两位小数的数字并不声称精确到两位小数 - 它声称您提供的输入按面值计算得出该数字。显示精度和现实世界中的准确性是不同的事情，将它们混为一谈是正确工具产生错误决策的最常见方式。

如果结果看起来有误，请在怀疑算术之前检查输入。实际上，绝大多数令人惊讶的答案都来自于以错误的单位输入的值、小数分隔符的读数与您预期的不同，或者应该更改的字段保留为默认值。这并不是对任何人的批评——这只是错误所在，知道先看哪里可以节省最多时间。

对比常见问题解答

什么是 WCAG 对比度？
普通文本：AA=4.5:1，AAA=7:1。大文本（18pt+）：AA=3:1，AAA=4.5:1。最大值为 21:1（黑底白字）。
为什么对比度很重要？
大约 8% 的男性有色觉缺陷。对于视力低下的人来说，在明亮的阳光下或在较差的显示器上，低对比度是很困难的。
对比度是如何计算的？
对比度 = (L1 + 0.05) / (L2 + 0.05)，其中 L1 和 L2 是相对亮度。使用带有伽马校正的 sRGB 公式。
