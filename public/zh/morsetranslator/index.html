在文字和摩斯电码之间互译

摩斯电码是历史上辨识度最高的编码系统之一。早在语音传输出现之前,那些独特的点和划就已经能跨越大陆和海洋传递信息。如今它仍然被用在业余无线电、航空导航、无障碍辅助设备,以及流行文化里。这个翻译工具能把任意文本瞬间转换成摩斯电码,把电码以音频形式播放出来,也能把摩斯电码解码回普通文字。

使用方法

在编码框里输入你的消息,摩斯电码译文会立刻显示在下方,字母之间用一个空格分隔,单词之间用斜杠分隔,完全符合摩斯电码标准书写惯例里点和划的表示方式。点击播放,就能听到直接在浏览器里生成的音频提示音,这是练习用耳朵而不是用眼睛识别摩斯电码的实用方法。反过来,把点和划粘贴进解码框,按照同样的字母和单词分隔规则,对应的普通文字会立刻显示出来。

摩斯电码的历史

塞缪尔·摩尔斯和阿尔弗雷德·维尔在19世纪三四十年代开发了摩斯电码,用于电报系统,通过长长的铜线把信号以脉冲形式发送出去。这套编码给每个字母和数字都分配了一种独一无二的短信号(点)和长信号(划)组合。最初的版本如今被称为美式摩斯电码,后来被精炼成国际摩斯电码,统一了时长比例并扩展了字符集,国际摩斯电码正是今天全世界通用的版本。

著名的SOS遇险信号——三个点、三个划、三个点(··· ─── ···)——之所以被选中,正是因为它足够简单、对称,即便在紧张状态下也很容易发送。

关于用耳朵学摩斯电码的小提示

熟练的摩斯电码报务员根本不会有意识地去数点和划的个数——练到一定程度,每个字母都会被瞬间识别成一种独特的节奏型,就像一个熟练的读者能一眼认出一整个单词,而不用一个字母一个字母地拼读一样。反复听这个工具播放的音频,从简短、常见的单词开始,是培养这种节奏识别能力远比死记一张点划对照表更有效的方法,这也是为什么资深的摩斯电码学习者几乎都会建议先训练耳朵,再训练眼睛。

摩斯电码是怎么运作的

每个字母、数字和常见标点符号都有一个独一无二的点划序列。字母E只是一个点,字母T则是一个划——英语里最常用的字母得到了最短的编码。字母之间的停顿相当于三个点的时长,单词之间的停顿相当于七个点的时长。正是这套时长结构,让摩斯电码既能靠声音传递,也能靠光信号或机械方式传递。

标准的时长比例是:

  • 点:1个单位
  • 划:3个单位
  • 同一个字母内部信号之间的间隔:1个单位
  • 字母之间的间隔:3个单位
  • 单词之间的间隔:7个单位

摩斯电码如今还用在哪里

世界各地的业余无线电爱好者至今仍在用摩斯电码进行远距离通信,尤其是在信号传播条件不佳的情况下,因为摩斯电码能穿透那些让语音通信完全无法进行的干扰。许多国家的业余无线电执照考试至今仍包含摩斯电码的内容。

航空领域用摩斯电码作为导航信标的识别方式:VOR(甚高频全向信标)和ILS(仪表着陆系统)发射台会定期用摩斯电码广播自己的识别码,让飞行员能确认自己使用的导航设备是正确的。

无障碍辅助技术有时也会用摩斯电码作为行动不便人士的输入方式。一个简单的按钮或开关就能发送点和划,让使用者能以熟练报务员才能达到的、出人意料的速度完成完整的文字输入。

这套编码为什么会这样设计

原始设计的巧妙之处,在于把编码长度和字母出现频率对应起来:因为塞缪尔·摩尔斯和阿尔弗雷德·维尔研究过英语文本中哪些字母出现得最频繁,所以最常见的字母E被分配了最短的信号——一个点,次常见的T得到了一个划。像Q、X、Z这类罕见字母则被分配了更长、更复杂的模式,因为整体目标是让传输普通文本的平均耗时最短,而不是让每个字母都同样快。这个原则——给更常用的符号分配更短的编码——在一个多世纪之后,又出现在了现代数据压缩算法里,让摩斯电码成了今天计算机科学中一些理念一个意想不到、却真实存在的先驱。

标点符号与专业信号

除了字母和数字,标准摩斯电码也为句号、逗号、问号等常见标点定义了编码,这也是为什么这个翻译工具能处理整句话,而不只是单个单词。业余无线电爱好者还会用一种叫"专业信号"的特殊组合序列来控制空中通话的流程——比如表示"消息结束"的一种序列,或者表示"邀请对方发送"的一种序列——这些序列会作为一整串不间断的点划发送,而不是拆成单独的字母,这是一层这个工具不会自动解码的约定,但如果你遇到了,知道它的存在还是有必要的。

Morse code FAQ

What is Morse code?
Morse code is a system for encoding letters, digits, and punctuation as sequences of dots and dashes (or short and long signals). It was developed in the 1830s and 1840s for use with the electrical telegraph. Each character has a unique dot-dash sequence.
How do I separate letters in Morse code?
Letters are separated by a single space. Words are separated by a slash / or three spaces. The decoder accepts both conventions.
Does the audio playback work?
Yes. The tool uses the Web Audio API to generate tones matching the dots and dashes. Dots are short tones, dashes are three times as long. Make sure your device volume is turned on.
