十进制数与罗马数字之间的互转

罗马数字已被使用了两千多年，至今仍广泛出现在时钟表盘、电影和书籍标题、超级碗、法律文件和纪念碑上。本转换器支持双向转换：输入十进制数字即可获得对应的罗马数字，输入罗马数字则可将其转回十进制。

罗马数字体系

罗马数字使用七个拉丁字母作为符号：I（1）、V（5）、X（10）、L（50）、C（100）、D（500）和 M（1000）。数字通过组合这些符号来表示，遵循两条主要规则：

加法规则：符号从大到小排列，数值相加。III = 1+1+1 = 3。VIII = 5+1+1+1 = 8。XIII = 10+3 = 13。

减法规则：当较小的符号出现在较大的符号之前时，减去其值。这样可以避免连续四个相同符号的情况。减法组合为：IV = 4（而不是 IIII），IX = 9（而不是 VIIII），XL = 40，XC = 90，CD = 400，CM = 900。

完整示例

  • MMXXIV = 2000+20+4 = 2024（2024年）
  • XLII = 40+2 = 42
  • CDXCIX = 400+90+9 = 499
  • MCMXCIX = 1000+900+90+9 = 1999
  • MMMDCCCLXXXVIII = 3888（最长的标准罗马数字）

罗马数字在今天的应用场景

钟表和手表：许多传统时钟表盘使用罗马数字表示小时。注意，出于视觉对称的考虑，时钟上的 IV 有时写作 IIII。

电影和电视：电影和电视节目的制作年份通常以罗马数字显示在片尾字幕中。续集电影也经常使用罗马数字：《洛奇 IV》、《星球大战 第四集》。

体育赛事：超级碗自第五届起一直以罗马数字编号。奥运会也使用罗马数字来标记届次。

王室和教皇称号：同名的君主或教皇以罗马数字区分：伊丽莎白二世、教皇约翰二十三世、亨利八世。

大纲和列表：正式大纲（一、二、三或 I、II、III）使用罗马数字作为主章节编号。

建筑和纪念碑：基石、纪念碑和公共建筑上的年份数字传统上以罗马数字刻写。

关于非标准写法的说明

并非你遇到的所有罗马数字都严格遵循本转换器所使用的减法规则。历史上，罗马人自己并不总是完全一致的，一些时钟表盘上至今仍然使用 IIII 而非 IV，纯粹是为了与对面的 VIII 取得视觉平衡，较古老的铭文中也可能使用纯加法的写法。本工具遵循的是标准现代规范——学校教学和出版、电影字幕及正式文件中使用的规范——始终采用最短的减法形式。如果你输入了非标准的写法（如 IIII），任何意外的结果都说明输入不是标准形式，而非转换本身有错。

罗马数字的局限性

罗马数字没有零的符号，无法表示分数，也不便于进行算术运算——用罗马数字做乘法非常繁琐。正因如此，这一系统逐渐被印度-阿拉伯数字的位值制（即 0-9 的数字系统）所取代。

如何使用本转换器

在第一个字段中输入十进制数字，罗马数字形式将即时出现；在第二个字段中输入罗马数字，则可将其转回十进制数。两个方向都会随输入实时更新，你可以自由地来回切换——输入 1999，看到 MCMXCIX 出现，然后稍作修改，观察十进制值随之变化。本转换器接受标准的减法罗马数字，覆盖从 1 到 3,999 的所有整数，这是古典体系的实际极限。

十进制转罗马数字的逐步过程

转换过程是反复匹配最大的可能罗马值，从剩余数字中减去它，然后继续。以 1994 为例：最大的可匹配值是 1000（M），剩余 994；然后 900（CM），剩余 94；然后 90（XC），剩余 4；然后 4（IV），剩余 0。按顺序读出符号就是 MCMXCIV。这种贪心的、每次取最大值的方法始终能产生正确的标准形式。

罗马数字转十进制的逐步过程

反过来，转换器从左到右读取罗马数字，累加每个符号的值，但如果一个符号后面紧跟一个更大的符号，则减去它的值。以 MCMXCIV 为例：M（1000）后面是更小的 C，所以加 1000；C（100）后面是更大的 M，所以减 100；M（1000）加上；X（10）后面是更大的 C，减 10；C（100）加上；I（1）后面是更大的 V，减 1；V（5）加上。合计 1000-100+1000-10+100-1+5 = 1994，精确匹配原来的数字。

为什么没有"零年"以及罗马数字没有零

一个值得了解的趣闻是，罗马数字体系完全没有零的符号——这个概念在体系建立时尚未被形式化，而罗马数字纯粹是为计数和记录数量而设计的，不具备位值运算的需求。这也是为什么历史历法直接从公元前 1 年跳到公元 1 年，中间没有零年，这一特殊性至今仍会在计算古代日期之间的精确跨度时引起混淆。

罗马数字常见问题解答

罗马数字的范围是多少？
标准罗马数字表示从 1 到 3,999 的数字。数字 4,000 需要带有不标准横线的符号 (M̄)。此计算器支持 1–3,999。
罗马数字符号是什么？
I=1，V=5，X=10，L=50，C=100，D=500，M=1000。减法表示法：IV=4、IX=9、XL=40、XC=90、CD=400、CM=900。
今天什么时候使用罗马数字？
罗马数字出现在钟面、书籍章节标题、电影续集 (Rocky IV)、超级碗编号 (Super Bowl LVII)、电影和纪念碑中的年份编号以及轮廓格式上。
